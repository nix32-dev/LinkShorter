# LinkShorter
  
Данная программа предназначена для сокращения ваших ссылок.  
  
Образец запроса (Формат JSON):   
```json
{
    "description":"описание",
    "link":"ваша ссылка для сокращения",
    "inpTimForm":"тип времени (поддерживаются: minute, hour, day)"
    "time":"целое число * на тип времени"
}
```

По умолчанию сервер стартует на ```:8080``` порту.  
Запросы на переход по корневому каталогу, любой тип подключения.  
Запросы на создание по каталогу ```/create``` с методом POST.  
Запросы на получение данных по каталогу ```/get``` с querry параметром ?info. (ALL/all/пустой ввод - получить информацию о всех ссылках или ввести значение (после "/") сгенерированной )  
Ссылки нужно указывать с ```http://``` или ```https://```
